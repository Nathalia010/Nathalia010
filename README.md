# 💼 Portafolio Web Personal

## 📖 Descripción

Este proyecto consiste en el desarrollo de un **portafolio web personal** diseñado para presentar de forma moderna, organizada e interactiva mi perfil profesional.

El sitio reúne información sobre mi formación académica, experiencia laboral, habilidades técnicas, proyectos y medios de contacto mediante una interfaz inspirada en un **Learning Platform Dashboard** de estilo oscuro. Todo el contenido se carga dinámicamente con **JavaScript**, permitiendo una navegación fluida sin necesidad de recargar la página.

## ⚙️ Arquitectura

Una de las principales características del proyecto es la **centralización de la información**.

Toda la información personal, académica, laboral, habilidades, proyectos y datos de contacto se encuentra organizada en objetos y arreglos dentro del archivo **`infoPersonal.js`**.

Gracias a esta arquitectura, basta con modificar un único dato para que el cambio se refleje automáticamente en todas las secciones del portafolio, facilitando el mantenimiento y la escalabilidad del proyecto.
